Counters

To add a counter after an item, prepend an EM with the class "Counter" to the LI:

<nav id="my-menu">
   <ul>
      <li><a href="/">Home</a></li>
      <li>
         <em class="Counter"></em>
         <a href="/about/">About us</a>
      </li>
      <li><a href="/contact/">Contact</a></li>
   </ul>
</nav>

If you're using a different classname, for example "bubble", you must specify this in the configuration.counterClass option:

$("#my-menu").mmenu({
   configuration: {
      counterClass: "bubble"
   }
});

If you want the counters to be added automatically to every item with a submenu, simply set the counters option to true: 

$("#my-menu").mmenu({
   counters: true
});
