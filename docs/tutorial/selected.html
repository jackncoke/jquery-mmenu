Selected item

To make an item visibly selected, add the class "Selected" to the LI:

<nav id="my-menu">
   <ul>
      <li class="Selected"><a href="/">Home</a></li>
      <li><a href="/about/">About us</a></li>
      <li><a href="/contact/">Contact</a></li>
   </ul>
</nav>

If you're using a different classname, for example "active", you must specify this in the configuration.selectedClass option:

$("#my-menu").mmenu({
   configuration: {
      selectedClass: "active"
   }
});
