Labels

To make an item appear as a label, add the class "Label" to the LI:

<nav id="my-menu">
   <ul>
      <li class="Label">Website</li>
      <li><a href="/">Home</a></li>
      <li><a href="/about/">About us</a></li>
      <li><a href="/contact/">Contact</a></li>
   </ul>
</nav>

If you're using a different classname, for example "title", you must specify this in the configuration.labelClass option:

$("#my-menu").mmenu({
   configuration: {
      labelClass: "title"
   }
});
